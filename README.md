# Mealify 🍽️

Mealify is a fully responsive restaurant landing page built with **pure HTML and CSS**.  
This project features **dark mode**, **animations**, and a **mobile-friendly sidebar menu**.

---

## 🚀 Features

- Fully **responsive design** (desktop, tablet, mobile)  
- **Dark mode toggle** using CSS  
- Smooth **CSS animations** and hover effects  
- **Mobile sidebar navigation** without JavaScript  
- **Grid-based gallery layout**  
- Contact section with **Google Maps embed**  
- Structured footer with **newsletter form** and social media links  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- Media Queries  
- CSS Grid & Flexbox  
- CSS Animations & Transitions  
- Google Fonts  
- Font Awesome Icons
